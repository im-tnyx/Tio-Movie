# Roadmap

This roadmap defines the recommended development phases for Tio-Flix.

## Phase 1: Foundation

- Project setup
- Clean package structure
- Theme setup
- Navigation setup
- Supabase config
- Environment configuration
- Basic error handling

## Phase 2: Authentication

- Splash session check
- Login screen
- Signup screen
- Forgot password screen
- Continue with Google
- Profile sync after login
- Logout

## Phase 3: Movie Data

- Movies table
- Categories table
- Movie detail data
- Poster and banner loading
- Home sections
- Search

## Phase 4: Player

- Media3 ExoPlayer setup
- HLS playback
- Fullscreen immersive mode
- Play/pause
- Seekbar
- 10-second forward/backward
- Resume playback
- Subtitles
- Audio tracks
- Quality selector

## Phase 5: Ads

- Google IMA SDK setup
- Pre-roll ads
- Mid-roll ads
- Ad break configuration
- Ad analytics events
- Fallback behavior when ad fails

## Phase 6: User Features

- Watch history
- Continue Watching row
- Favorites / My List
- Profile screen
- Recently watched

## Phase 7: Admin & Backend

- Admin movie upload flow
- Category management
- Ad break management
- Reports and analytics
- Secure backend actions

## Phase 8: Release

- QA testing
- Crash reporting
- Performance optimization
- Play Store assets
- Privacy policy
- Production build

## Future features

- Offline downloads
- DRM
- Parental controls
- Multi-profile support
- Push notifications
- Recommendation engine
- TV app support
