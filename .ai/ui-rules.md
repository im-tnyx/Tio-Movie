# UI Rules

## Visual direction

Tio-Flix should feel like a premium OTT app.

- Dark theme first
- Large posters and banners
- Clear typography
- Smooth navigation
- Fast perceived loading
- Clean player controls

## Mobile UI

Mobile should support:

- Touch navigation
- Bottom navigation or simple top-level navigation
- Portrait browsing
- Landscape fullscreen player
- Gesture-friendly player controls

## TV / Fire TV UI

TV and Fire TV should support:

- D-pad navigation
- Visible focus state
- Large cards
- Large text
- Simple layouts
- Remote-friendly player controls
- Back button behavior

## Do not do this

- Do not reuse tiny mobile spacing directly on TV.
- Do not make TV controls touch-only.
- Do not hide focus state.
- Do not block the user in infinite loading states.
