# Project Context

Tio-Flix is an Android Kotlin OTT/movie streaming app project.

## Product direction

Build a Netflix/SonyLIV/Prime Video style app experience for:

- Android Mobile
- Android Tablet
- Android TV / Google TV
- Amazon Fire TV / Fire TV Stick

## Core product features

- Email/password auth
- Continue with Google
- Movie home screen
- Search
- Movie detail page
- HLS video playback
- Netflix-style player controls
- Pre-roll and mid-roll ads
- Watch history
- Continue Watching
- Favorites / My List
- TV and Fire TV remote-friendly navigation

## Recommended stack

```text
Android: Kotlin + Jetpack Compose
Architecture: MVVM + Repository Pattern
Auth: Supabase Auth
Database: Supabase PostgreSQL
Streaming: Bunny Stream / Mux
Player: Android Media3 ExoPlayer
Ads: Google IMA SDK / Google Ad Manager
```

## Legal boundary

Only legal/licensed content should be used. Do not add scraping, piracy, unauthorized streaming links, DRM bypass, or ad bypass logic.
